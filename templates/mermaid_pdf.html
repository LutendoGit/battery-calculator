Diagram Input

Clear

What It Accepts

  • Mermaid headers like graph TD and graph LR.
  • Nodes in the form A[Label].
  • Solid arrows using --> and dashed arrows using -.->.
  • Multi-line labels using <br/> inside node labels.
  • Optional edge labels such as A -->|Backup| B.

Quick Example

graph TD A[Solar Panels<br/>306 kW] --> B[MPPT Controllers<br/>300 kW] B --> C[Battery Bank<br/>6,120 kWh LiFePO4] C -.-> D[Backup Path]

How It Works

  • The app parses the Mermaid text into nodes and edges.
  • It auto-generates a readable layout for the PDF.
  • The resulting file is downloaded directly to your machine.
